Bu layihə HTML, CSS və JavaScript əsasında hazırlanmış süni intellekt dəstəkli PowerPoint slayd generatorudur. Tətbiq birbaşa brauzerdə işləyir və istifadə üçün əlavə server və ya mürəkkəb qurulum tələb etmir. İstifadəçi mövzunu və slayd sayını daxil etdikdə sistem süni intellekt API-si ilə əlaqə qurur və təqdimat üçün uyğun mətnlər yaradır. Yaradılan mətnlər ardıcıl olur, mövzu daxilində məntiqi axın qorunur və hər slaydın məqsədi aydın şəkildə ifadə edilir.
Layihənin əsas məqsədi slayd hazırlama prosesini sadələşdirmək və bu prosesi avtomatik hala gətirməkdir. JavaScript tərəfi istifadəçidən daxil edilən məlumatları emal edir və Fetch API vasitəsilə AI sisteminə göndərir. AI-dən gələn cavab strukturlaşdırılır və slayd məntiqinə uyğun bölünür. Hər slayd üçün ayrıca başlıq və qısa izahlı mətn formalaşdırılır ki, təqdimat həm oxunaqlı, həm də anlaşılan olsun.
Tətbiq 1-dən 100-ə qədər slayd yaratmaq imkanına malikdir və bu da onu həm qısa, həm də geniş mövzulu təqdimatlar üçün uyğun edir. Kod sadə və səliqəli yazıldığı üçün layihəni başa düşmək və üzərində dəyişiklik etmək asandır. Gələcəkdə dizayn temalarının əlavə edilməsi, şəkillərin avtomatik yerləşdirilməsi və çoxdilli dəstəyin inteqrasiyası mümkündür.
Bu layihə əsasən öyrənmə və praktiki təcrübə qazanmaq məqsədi daşıyır. API açarının JavaScript faylında saxlanılması təhlükəsizlik baxımından real məhsullar üçün uyğun hesab edilmir. Buna baxmayaraq, mövcud quruluş frontend əsaslı AI tətbiqlərinin iş prinsipini anlamaq üçün güclü əsas yaradır. Layihə düzgün inkişaf etdirildikdə funksional və istifadə oluna bilən bir məhsula çevrilə bilər.
